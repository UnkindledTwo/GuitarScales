# GuitarScales
A Guitar Scale App Built With Qt

## Adding Custom Scales
Create scales.json in the same directory as the executable.

### Example scales.json
``` json
[
    {
    "Name": "Major",
    "Intervals": ["1", "2", "3", "4", "5", "6", "7"]
    },
    {
    "Name": "Minor",
    "Intervals": ["1", "2", "b3", "4", "5", "b6", "b7"]
    }
]
```
